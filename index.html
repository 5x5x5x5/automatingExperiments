Autoprotocol

Autoprotocol is not a programming language. Autoprotocol-encoded protocols contain no logic and are guaranteed to complete in a fixed amount of time. Autoprotocol can be generated by programming languages and consumed by programs of all kinds, but Autoprotocol itself is just a data standard. You can do very sophisicated things like autonomous directed evolution or running self-healing assays using code that generates Autoprotocol, but such logic is "external" to the protocol itself.
